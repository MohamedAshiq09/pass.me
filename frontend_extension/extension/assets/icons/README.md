# Extension Icons

Place the following icon files in this directory:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

These icons will be used for the browser extension toolbar and Chrome Web Store listing.

You can create these icons using any image editor or online icon generator. The icons should represent the Pass.me brand with the lock/key theme.
